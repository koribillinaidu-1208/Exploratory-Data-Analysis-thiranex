# 🚢 Titanic EDA Project

## 📌 Objective
Perform Exploratory Data Analysis (EDA) on Titanic dataset to uncover patterns and trends.

## 📂 Dataset
Dataset contains passenger details such as age, gender, class, fare, and survival.

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Steps Performed
- Data Cleaning
- Handling Missing Values
- Data Visualization
- Correlation Analysis

## 📊 Key Insights
- Females had higher survival rate
- 1st class passengers survived more
- Fare positively influenced survival
- Age had minimal impact
